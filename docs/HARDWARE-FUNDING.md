# Hardware Testing Tracker

The currently released workflow supports only the original Lenovo Legion Go / Legion Go 1 hardware identity documented in this repository.

A device is not supported unless its status explicitly says **Released**.

## Status definitions

| Status | Meaning |
|---|---|
| Proposed | Community interest recorded; no active acquisition plan |
| Funding | Funds are being collected toward the exact device |
| Acquired | Exact device is physically available for testing |
| Research | OEM packages, hardware IDs, recovery, and driver behavior are being analyzed |
| Validation | The workflow is undergoing full reboot-to-reboot validation |
| Released | Device-specific workflow passed the complete final audit |
| Paused | Work cannot currently continue |
| Not feasible | Safe or maintainable public support was not practical |

## Current hardware

| Device | Status | Note |
|---|---|---|
| Original Lenovo Legion Go / Legion Go 1 | Released | Public Beta v2.1 is current; validated target AMD 26.6.4 |

## Additional targets

No additional handheld target is formally active.

Ryzen Z1 Extreme branding alone does not prove shared device IDs, OEM extensions, firmware behavior, metadata, power controls, sensors, or recovery requirements. Each model requires separate physical testing and a complete validation cycle.

## Funding boundary

Donations can fund access to hardware and testing resources. They do not guarantee support, a successful technical result, or a release date.
